# Hades Mod Manager Documentation

Welcome to the Hades Mod Manager documentation. This guide provides comprehensive information about installing, configuring, and using the Hades Mod Manager to enhance your Hades gaming experience with community-created mods.

## Overview

Hades Mod Manager is a desktop application built with React and Electron that simplifies the process of managing mods for the game Hades. It provides an intuitive graphical interface for installing, enabling, disabling, and organizing mods without requiring manual file manipulation.

## Key Features

- **Mod Management**: Install, enable, disable, and uninstall mods with a single click
- **Mod Detail View**: View comprehensive information about each mod including description, author, and version
- **Mod Downloader**: Download mods directly from supported sources
- **Log Viewer**: Monitor mod loading and application activity through detailed logs
- **Action Bar**: Quick access to common operations

## System Requirements

### Minimum Requirements
- **Operating System**: Windows 10 or later
- **Storage**: 100 MB free space
- **RAM**: 4 GB
- **Hades Game**: Installed and configured

### Recommended Requirements
- **Operating System**: Windows 11
- **Storage**: 500 MB free space
